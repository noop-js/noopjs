# @noopjs/runtime

Browser runtime for NoopJS. Provides DOM binding primitives used by compiled components:

- `bindText` — reactive text nodes
- `bindAttribute` — reactive attributes
- `bindStyle` — reactive inline styles
- `bindEvent` — delegated event handling
- `onMount` / `onUnmount` / `onUpdate` — lifecycle hooks
- `createContext` / `useContext` — context API
- `Suspense` / `ErrorBoundary` — async and error boundaries
- `createPortal` — teleport to other DOM nodes

Part of the [NoopJS](https://github.com/noop-js/noopjs) framework.
