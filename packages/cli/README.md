# @noopjs/cli

Command-line tools for NoopJS projects.

```bash
noopjs dev          Start dev server
noopjs build        Build for production
noopjs preview      Preview production build
noopjs generate     Scaffold components and pages
noopjs analyze      Bundle size report
noopjs check        Type-check .noop files
noopjs init         Initialize a new project
```

Part of the [NoopJS](https://github.com/noop-js/noopjs) framework.
